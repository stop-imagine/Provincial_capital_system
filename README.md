# Provincial_capital_system
安卓app实现中国各个省会信息（美食、景点、高校）的轮播功能
项目总体功能基本实现，包括查询，增删改，以及学习轮播，轮播速度修改。不足在于页面设计不够细致，但总体较完整。以下展示部分功能页面截图供参考。
